# Student Result Management System

A simple console-based Python application to manage student results using file handling.

## Features
- Add student name and marks
- Calculate total, percentage, and grade
- Store student data in a text file
- View all student records

## Technologies Used
- Python
- File Handling

## How to Run
1. Clone the repository
2. Run `main.py` using Python
3. Use the menu options to add or view students
